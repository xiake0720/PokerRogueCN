# HUD V2 素材包

来源：用户提供的 2048×1152 透明 UI 素材图。

本次只新增切割后的运行时素材，不修改场景与脚本。

目录：

- `assets/ui/runtime/panels/`：HUD 主框、信息条、牌架、弹窗和数值底板
- `assets/ui/runtime/buttons/`：橙、绿、红、蓝按钮底图
- `assets/ui/runtime/icons/`：菜单、设置、导航与状态图标
- `assets/ui/runtime/tokens/`：金币、筹码与进度点
- `assets/ui/runtime/cards/`：卡牌示例和牌背

左侧 HUD 重构首选素材：

- `panels/hud_v2_panel_full.png`：按参考图二排布好的完整空白 HUD 框

- `panels/hud_v2_surface_green.png`：深绿色纹理内容底板
- `tokens/hud_v2_icon_coin.png`
